# 🚀 30 Days DSA Challenge

Welcome to my **30 Days DSA Challenge** repository.

In this challenge, I will practice **Data Structures and Algorithms daily** and upload the problems along with their solutions here.

The goal is simple:
👉 Stay consistent and improve problem-solving skills.

---

## 📅 Challenge Plan

For the next **30 days**, I will:

* Solve multiple DSA problems every day
* Focus on different topics (Arrays, Strings, Trees, Graphs, etc.)
* Write clean and optimized code
* Improve problem-solving speed and logic
